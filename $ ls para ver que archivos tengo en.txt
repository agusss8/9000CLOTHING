$ ls para ver que archivos tengo en la direccion que estoy ahora

$ cd para entrar a una carpeta

$ cd .. para volver para atras

$ clear para limpiar consola

$ git init para inicializar un repositorio en la carpeta donde estoy

$ git config user.email/user para ver quien el registro de qn estubo

$ git add . para aniadir y seleccionar al buffer mi pedaso de archivo

$ git commit -m "comentario"  para aniadir al repositorio y que este fijado en la nube

$ git push -u origin main para subirlo finalmente a git a mi repositorio online

$ git status para saber el estado de mis archivos y ver si estan o no en la nube
